Found via targeted search (not part of the original leaf-protein-extraction search clusters) after the project's owner directly asked whether adjacent AI/ML literature exists for this kind of sensing problem. Key value: establishes that (1) e-nose arrays combined with ANN/deep learning classifiers reliably detect off-flavor in a real food system (catfish) at accuracy exceeding human olfactory sensitivity, and (2) this is recognized as a "hot research area" per the 2025 review, with deep learning actively displacing classical multivariate methods (PCA/LDA) for this exact task. Neither source is about leaf protein or plant-based systems - this is precedent for the SENSING+AI METHODOLOGY, not for the specific application. The gap this project would fill is applying this proven methodology (or an analogous one, e.g. DGMA-based real-time liquid-phase sensing) specifically to leaf-protein off-flavor optimization, which does not appear to exist in the literature searched so far.
